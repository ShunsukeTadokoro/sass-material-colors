# Sass Material Design Color Pallete
From https://www.google.com/design/spec/style/color.html

## Usage

``` scss
@import 'material-colors'

.btn {
  color: $m-grey-50;
  background: $m-teal-a700;
  box-shadow: 0 3px 0 darken($m-teal-a700, 6%);
  ...
}
```
